Detailed Description

Provides a list of tokens to ignore when auto-detecting common name tokens.

The module exposes two helpers:

  • defaultIgnoreTokens() returns a built-in, hardcoded list of tokens.
  • getIgnoreTokens() attempts to load additional tokens from data/ignoreTokens.json (key ignoreTokens) and falls back to the built-in list when the file is missing or invalid. Tokens are normalized to lowercase and cached on first access.

const std::vector< std::string > & getIgnoreTokens ( )
inline

Returns the ignore tokens loaded from data/ignoreTokens.json or a fallback.

On first call this function attempts to open data/ignoreTokens.json and read an array at key ignoreTokens. If the file is missing, malformed, or does not contain the expected array, the function falls back to defaultIgnoreTokens().

The loaded list is normalized to lowercase and cached for subsequent calls. The function is not synchronized for multi-threaded access; if the program uses threads it should ensure external synchronization when calling this function during startup.
