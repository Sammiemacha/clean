Detailed Description

Main entry point and terminal user interface (TUI) for the file organization utility.

This module provides the primary menu system and navigation logic for organizing files by type or by common name patterns. It handles user input validation, menu rendering, and delegates to specialized cleaning functions.

void byName ( const fs::path & directoryPath)

Provides the Terminal User Interface (TUI) for cleaning files by name.

Terminal User Interface for organizing files by name.

Parameters
directoryPathThe filesystem path to the target directory.

Presents a submenu similar to byType() but for name-based organization. Options include:

  • Return to main menu (0)
  • List all files in the directory (1)
  • Execute the file cleaning operation (2) - organizes into name-based subdirectories
  • Change to a different directory (3)

The function is recursive and will re-display the menu after each operation.

See also

Display the byName submenu with numbered options for user selection. Options 0-3 are validated in the loop condition.

◆ operation()

void operation ( )

Displays the main operation menu and handles user selection.

Main terminal user interface (TUI) operation loop.

Presents a menu with options to:

  • Exit the program (0)
  • Clean files by type (1)
  • Clean files by name (2)

Validates input and loops indefinitely until user chooses to exit.

The function handles input errors gracefully, clearing the input buffer and prompting the user to retry on invalid selections.
