Detailed Description

ANSI terminal color constants used by the CLI.

This header defines a small set of ANSI escape sequence string constants for coloring and styling console output. They are intended for use in simple terminal UIs and logging messages.

Note
These codes are standard on most UNIX-like terminal emulators. On Windows consoles prior to Windows 10 the escape sequences may not render correctly without additional configuration.
